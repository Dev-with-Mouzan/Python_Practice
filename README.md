# Python Practice Programs

This repository contains a collection of Python scripts created to practice basic programming logic and coding concepts. Each script focuses on a specific problem or algorithm.

## List of Programs

| File Name | Description |
| :--- | :--- |
| `Even_odd.py` | Determines whether a given number is even or odd. |
| `Swap.py` | Swaps the values of two variables. |
| `celsius_to_fahrenheit.py` | Converts temperature from Celsius to Fahrenheit. |
| `count_vowels.py` | Counts the number of vowels in a given string. |
| `factorial_loop.py` | Calculates the factorial of a number using a loop. |
| `fibonacci_series.py` | Generates the Fibonacci series up to a specified term. |
| `gcd.py` | Finds the Greatest Common Divisor (GCD) of two numbers. |
| `input.py` | Demonstrates basic user input and output in Python. |
| `largest_number.py` | Finds the largest number among a set of values. |
| `lcm.py` | Finds the Least Common Multiple (LCM) of two numbers. |
| `leap_year_check.py` | Checks if a specified year is a leap year. |
| `number_guessing_game.py` | A simple game where the user guesses a random number between 1 and 100. |
| `palindrome_check.py` | Checks if a string or number is a palindrome. |
| `primes_1_to_100.py` | Lists all prime numbers between 1 and 100. |
| `reverse_string.py` | Reverses a given string. |
| `sum_n_natural_numbers.py` | Calculates the sum of the first N natural numbers. |
| `sum_of_digits.py` | Calculates the sum of the digits of a given number. |
| `table.py` | Prints the multiplication table for a given number. |
| `vowel_consonant_check.py` | Checks if a character is a vowel or a consonant. |
